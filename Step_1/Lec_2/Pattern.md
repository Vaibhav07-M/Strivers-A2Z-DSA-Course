This repository contains 22 different pattern printing programs implemented in C++. These are fundamental programming exercises that help build logic and understanding of nested loops.

## Code Structure

Each pattern is implemented as a separate function:
- `pattern1(int n)` through `pattern22(int n)`
- Each function takes an integer `n` as input for the size of the pattern
- The `main()` function demonstrates all patterns with `n = 5`

## Learning Objectives

These patterns help develop:
- Understanding of nested loops
- Logic building skills
- Pattern recognition
- Problem-solving abilities
- Control flow mastery
